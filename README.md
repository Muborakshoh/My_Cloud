# File Repository

A modern, client-side file repository application that allows you to organize and preview your files directly in the browser.

## Features

- Upload and organize multiple file types (images, music, videos, documents)
- Real-time image and video previews on hover
- File categorization and filtering
- Search functionality
- Responsive design for all devices
- Local storage for persistent data
- Preview support for images, audio, and video files

## Usage

1. Click "Upload Files" to add your files
2. Use category buttons to filter files by type
3. Use the search bar to find specific files
4. Hover over images and videos for quick previews
5. Click on any file to open full preview
6. Files are automatically saved in your browser's local storage

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Font Awesome for icons
- Local Storage API for data persistence

## Setup

Simply clone the repository and open `index.html` in your web browser. No server or build process required!

```bash
git clone [repository-url]
cd repository
# Open index.html in your browser
```
